# ============================================================================
# VERSION CONSTRAINTS
# ============================================================================
# Upper bounds prevent breaking changes from newer package versions:
# - diffusers<0.32.0: VAE API changes in 0.32+
# - accelerate<1.0.0: API restructuring in 1.0
# - huggingface-hub<1.0.0: Breaking changes in 1.0
# - numpy<2.0.0: ABI incompatibility with PyTorch <2.3.1
# - gradio<5.0.0: Breaking changes (removed params, SSR mode, proxy issues)
# - pydantic<2.11.0: Bug in gradio_client with pydantic 2.11+ causes TypeError
#
# To upgrade to newer versions, test thoroughly in a separate environment first.
