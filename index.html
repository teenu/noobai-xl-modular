About 4CGT

4CGT is an open-source anime image generation tool built on NoobAI XL V-Pred with DoRA anatomical correction, 600-token prompt support, and cross-platform deterministic output.

Every image on this site was generated with 4CGT. The prompts, seeds, and settings are published so anyone can reproduce them exactly.

Source code: github.com/teenu/4cgt

Character names, designs, and associated properties belong to their respective copyright holders. Images on this site are AI-generated fan art for personal, non-commercial showcase. Donations support 4CGT open-source software development, not character artwork.
