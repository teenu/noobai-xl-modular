Deterministic Output

Same seed, same image hash — macOS, Windows, Linux. CPU RNG, enforced deterministic algorithms, cross-platform parity.

Capabilities

2D → 3D Pipeline

Integrated Apple Sharp image-to-3D Gaussian Splat conversion. Sub-second inference with optional camera-trajectory video render.

ControlNet Pose

OpenPose skeleton conditioning optimized for v-prediction models. Higher conditioning scales for proper pose adherence.

DoRA Stabilizers

Three modes: None (optimal defaults), Optimized (preset-locked 34-step schedule), and Manual (per-step binary control).

Web interface with character/artist autocomplete from Danbooru and e621. Full CLI for batch generation and scripting.

Broad GPU Support

RTX 5090 (Blackwell) to GTX 1060. Apple Silicon native. Automatic CPU offloading for GPUs under 8GB VRAM.
